## 第二章 游戏世界

“欢迎来到主神，这里有着您想体验的一切！”

一句带着些微电音的合成AI语音提示，代表着这个游戏被正式启动。

“检测到您的身份信息，已自动为您切换为中文，如若需要更改，请在左上角选择。”

紧接着就是一个大大的加号显示在了我的面前，我调用我的意识点击了加号，出现了一个选项，让我选择游戏世界的总体背景，看了看其他地方，并没有其他选项。

正当我疑惑的时候，出现了一个AI声：“如若您不善于选择或者对于细节并不是很在乎，则仅需选择带着星号的必选项，填好必选项后即可点击下方的开始生成按钮，剩余的选项游戏AI会自动帮您调整至您所倾向的世界。”

“啊，原来真的是这样啊，我还以为虚假宣传呢，吓死我了”，我内心悬着的心放了下来，于是我就一个选项一个选项地选了下去。

> 游戏主体背景：中世纪 -> 魔法世界 -> 存在类人种族 -> 存在神明（不分正邪） -> 自动补充
>
> 职业分类：战士、骑士、术士、法师、弓箭手、刺客、策士
>
> - 特殊设定：能够施法者必须拥有特殊的魔法基因，且需要进行魔力激活
>
> 职业介绍：介绍各职业的从属关系
>
> - 战士：物理伤害为主，体魄强悍，可使用基础的为武器附魔的魔法
>
>  - 骑士：战士的荣誉称号，足够强大后可自主前往骑士工会进行资格测试
>  - 术士：平庸的法师，没有过高的天赋和实力，但是魔法基因一般超出其他职业
>  - 法师：经过魔法天赋认证或达到一定实力的术士
>  - 弓箭手：以弓箭为主的远程物理输出职业
>  - 刺客：隐藏在黑暗中的暗杀者，实力壮大后吞并了盗贼工会
>  - 策士：高智商人士，多为贵族附属，在军队中任职军师或为家族发展出谋划策
>
> 游戏背景时间设置：世界已存在3亿年 -> 莽荒时代（世界法则出现）-神明时代（世界法则孕育原初神明）-创生时代（神明创造各种种族）-硝烟时代（各族文明争夺大陆统治权）-近现代（出现种族灭绝，神明死亡）-人类纪元（现如今由人类主导的世界）
>
> 大陆设定：自动生成
>
> 世界主导种族设定：人类 -> 人种不限 -> 命名方式（中西混合） -> 种族存在形式：国家制 -> 遍布大陆各地 -> 文化背景依托于历史与地理环境自动生成
>
> 玩家所处时间：人类纪元2000年（距离大混战仅仅过去2000年，各族间矛盾依旧尖锐）
>
> 玩家所处位置：人类皇城 -> 所属于霍顿公国 ->其余国家名称由系统生成
>
> - 霍顿公国设定：种族主义泛滥（排斥其他类人种族） -> 奴隶制度存在 -> 第二强盛的人类公国 -> 所处沿海地区 -> 经济贸易发达 -> 军事强盛
> - 公国势力分布：皇族、魔法协会、生命教会、航海工会、商人联盟、各职业工会（其中以战士工会为主）
> - 公国贵族阶层：皇族 -> 皇族近亲 -> 册封大贵族 -> 皇族远亲 -> 公爵 -> 皇族附属 -> 侯爵 -> 伯爵 -> 子爵 -> 男爵
> - 公国周边环境：两面环山，一面环水，易守难攻
> - 公国实力：最强盛的魔法国度，拥有世界上最强的人类魔法师
>
> 玩家所属机构：皇城初级魔法塔 -> 所属霍顿公国魔法协会 -> 高层4/5为霍顿公国皇族剩余1/4为霍顿公国皇族附属
>
> 主城魔法塔情况介绍：
>
> - 顶级魔法塔(全世界仅此一座)：处在城中心
> - 高级魔法塔：主城有3座，呈三角形分布在内城中段，中点为顶级魔法塔
> - 中级魔法塔：主城有8座，呈环状分布在高级魔法塔外层
> - 初级魔法塔：主城有16座，呈环状分布在中级魔法塔外层，处于内城边缘，是外城与内城的分界线
>
> 玩家职位：魔法学徒，兼职初级魔法塔教室女仆长（常年穿着女仆装）
>
> 职业信息（女仆）：所有负责魔法塔后勤以及接待的女性统称
>
> + 特殊设定：魔法塔仅有女性后勤，女仆职位并非没有战斗力，仅仅是为不想参与战斗的女性魔法师提供了一个工作岗位，其中多为魔力研究者，不同等级的魔法塔均设有女仆职业
>
>   > 普通女仆在身份上和普通魔法学员没有区别，仅仅是一种职业
>
> + 女仆装：由魔法塔提供，根据身份等级的不同会增设不同的标识
>   + 女仆装等级标识：由发带进行区分，等级标识由魔法塔直接管理，高级魔法师亲自制作，无法被重新染色且指定本人特殊精神频率才可佩戴成功
>     + 女仆长：<font color=#8F786F > 棕色发带 </font> 
>     + 正式女仆： <font color='white' >白色发带 </font>
>     + 见习女仆：<font color=#66CCFF > 浅蓝色发带 </font>
>     + 贴身女仆：<font color=#FF7F50 > 浅红色发带 </font>
>     + 育婴女仆：<font color=#FFDEAD > 米黄色发带 </font>
> + 女仆魔法：仅需要拥有魔法基因即可，最基础的魔力运用
>
> - 女仆长：直属于魔法塔高层，多为魔法学徒且不止一位，各自管理不同的部门，当女仆长因学业或其他原因无暇管理时，魔法塔会从其下属女仆中选择一位进行职位交替
>
> - 大厅女仆：负责日常的垃圾清理和接待事项
>
> - 厨房女仆：负责魔法塔日常的一日三餐，食材与佐料的采购、管理和储存等事项
>
> - 教室女仆：某个特定教室的管理者，在课上通常会作为老师的助手，并负责管理本教室所需要用到的魔法器材
>
>   > 某些常识课和小孩子的基础教育课程也会由相应的教室女仆负责
>
> - 育婴女仆：特殊女仆职业，负责帮助魔法塔中因工作或实验问题可能长时间无法照顾孩童的正式魔法师照顾孩子，通常由魔法师直接指定魔法学徒
>
> - 贴身女仆：高级魔法师所拥有的女仆，一般是原女仆长担任，由魔法师直接指定
>
>   > 此职业由高级魔法塔直接管理，不可强行征收，需被指定人同意
>
> - 见习女仆：由激活了魔法天赋的女性魔法学徒报名参加，报名者会额外开设女仆课程供其学习当前魔法塔各项管理事宜，期末考试会根据各项考核分数被分入相应职位
>
> - 高级魔法塔的女仆职业多为下属低级魔法塔的学员晋升后担任，同样也可以由女仆长向魔法塔申请让魔法天赋高的低级魔法师担任
>
> 玩家身份设置：猫人族 -> 女性 -> 少女时期 -> 魔法师学徒（主修冰系） -> 魔法等级为初级巅峰（进入游戏后会自动生成魔力等级分级） -> 家庭背景：战争孤儿，由族内长老抚养长大 -> 具有很高的魔法天赋 -> 固有天赋：法则以下最强伪装术（魔力极低或为0时时才可被看破伪装）
>
> 玩家所属种族设定：母系社会 -> 议会制 -> 避世种族 -> 种族属性：魔法亲和（拥有超过一般种族的魔法元素亲和度） -> 族内拥有高端战力 -> 信奉魔法女神
>
> 玩家名称：青凰（人类形态），羽笙（猫人族形态）
>
> 玩家形象：见结尾图
>
> 天文历法：中世纪水平
>
> 度量衡：现代化刻度标准
>
> 时间概念：现代24小时制
>
> 制作工业水平：可量产基础的火枪和火炮，其中以矮人的品质最高，精灵族的颜值最高
>
> 

“砰砰砰~”，正想着接下来的设定要怎么写的时候，头盔被敲响了。

“小猫咪，起床了，时间到了，要去上课了，别忘了今天点名的。”

我看了一眼右下角的时间，竟然已经两点了，时间过的好快，看来来不及构思剩下的设定了，不过主要设定也已经选完了，接下里就交给AI自己生成吧。

“来了来了，等下我。”，我赶紧点了生成世界的按钮，把头盔从头上摘了下来，捋了捋自己的长发，一只手扎着辫子，一只手拿起早就放在枕头旁边的书和笔，从赶忙上铺跳了下来（大学宿舍上床下桌），跟上了室友的脚步去教室。

这节课是数据结构，来到教室之后选了靠前的座位，随后班上的同学有的三三两两的，有的以寝室为单位赶到了教室，两点半，课程准时开始。

“同学们，今天我们讲顺序表，请大家把课本翻到xx页，所谓顺序表呢，上节课提到过，其实就是……”

认真听讲的必修课总是过的很快，一不留神就过去了两个小时的上课时间，紧接着就是换教室，去上下一门课，一门公共选修课，名字叫“中国酒文化”，毕竟只是为了凑学分的选修课，这门课其实就很轻松了，整个课堂也是在很放松的情况下度过的，老师也和我们分享了很多有趣的事情，比如“人民都以为品酒师很能喝酒，然而实际上都是酒是不咽下去的”，“茅台酒厂的独特味道是因为其产地的生态环境而不是工艺技巧”，“一些低价酒的价格甚至都低于出厂价还标榜自己绝对是正品”，以及保证了最后一节课会带酒过来给我们品尝。

愉快的一下午就在这样的课程中结束了，我和三个室友在吃完了例行的晚饭之后也回到了宿舍（新生抢饭总是非常积极的），在为了庆祝度过了一周所有课时而举办的周常的餐后魔女茶话会上，我们彼此交流了这周遇到的事情和作业，当然这其中也少不了我们对于“主神”的讨论，我们彼此交换了大家所创建的世界是个什么样子的。

羽风首先说了自己创建了一个猫娘游乐园，她想要体验一下被猫娘包围的感觉，说实话，我顿时感觉到了满满的恶意。毕竟之前的打赌赌约她就说过要我扮猫娘一天，但是被我严词拒绝了，宿舍里面玩玩还好，她竟然让我穿着cos装在校园里面正常过一天，实在是太让人害羞了。

羽陌则是创造了一个武侠世界，她是个比较传统的武侠迷，虽然现在的科技已经发展到了只要体质足够，经过了训练就能去体验馆体验武侠世界了，但是她还是想去试一试自己的武侠世界。

欧阳倩倒是挺出乎我们的意料的，她创造了一个21世纪的现代战争世界，她作为特种兵将会上战场。

轮到我了，我在听完了她们说的内容之后倒是说了我创建了一个魔法世界，随后在我描述具体设定的时候，羽风注意到了我描述的内容中有一点特别的。

“停停停，小猫咪，我注意到一个事情，你说你的世界里面也有猫娘这个种族，那你在游戏里选了一个什么种族啊？”羽风一针见血地问了出来。

“……，为什么你会关注这个？”我有点疑惑地问道。

“我下午选修课在论坛上看到的心理分析，当一个种族的设定过于详实且背景十分强大的时候，往往会被作为玩家自己的种族，毕竟背靠大树好乘凉”。

“我，我当然是选的人族了，这还用问吗？”我有点心虚地说道。

“是吗？世界生成之后我们可是能看到好友创建世界的词条的哦，不要撒谎！”羽风靠了过来，说出了我担心的事情。

“为什么会有这种事情啊？还能看到世界词条的吗？”我疑惑地问道。

“那当然了，为了方便别抄写好的词条，官方会默认公布玩家所创建的世界所用到的词条的”羽风一脸怀疑地看着我，“你该不会真的选了猫娘吧？”

“我，我……”我有点语无伦次。

“果然，你还是愿意做猫娘的，那为什么不答应我之前的那个赌约啊？”羽风有点气鼓鼓的，质问着我。

“我，不对，谁要在学校里这么干啊，我要是真的接了我不就闻名全国了吗？”我的脑子有点没转过来弯，差点被她绕进去了。

“我不管，你快点过关，我到时候要去你的世界里享受你的服务！”此时的羽风仿佛挂在了我的身上，她坐在我的左边，双手搭在我的右肩上，整个身子附在了我的身上，6头则靠在了我的左肩上，我都能很明显地感觉到我的左手手臂被两团软软的东西顶住了。

“好吧好吧，我尽快。”眼见我无法逃脱被识破的命运，我也就干脆直接放弃了辩解。看了眼仿佛一只‘真·猫娘’一样“挂在”我身上的羽风，我无奈地说道，“倒是你那个‘猫娘乐园’一样的世界能生成什么任务啊，我倒是挺好奇的。”

“商业啊，我在游戏里要保证我开设的店铺营业额是多少，我点了创建世界之后AI给出的判定，我还得去现学商业的内容，好烦啊，打个游戏还要学习……”，羽风顿时泄气了。

“啊？AI会给任务的吗？我中午走的有点匆忙，没注意啊，会不会给我生成什么究极难的任务啊？”我有点慌神了。

“我怎么知道啊，反正我看论坛上面总结的，大概就是到达某个实力阶段或者带领母族到领导层之类的了。”羽风趴在我的耳朵边说道。

“那我待会得去看看到底是个什么任务了，得提前做点心理准备。”说完之后我就爬上了自己的床，戴上了头盔进去看了看。

____

"欢迎来到主神世界！"

伴随着中午才听过的机械女声，我进入了主神，查看了我那个正在生成的世界。

世界任务：达到半神等级并将猫人族由隐世带入霍顿公国管理层。

下面还有一句提示：您可以为您的世界命名并添加介绍。

这个提示对于我这种的起名困难症有点小挑战，不过最终我还是起了一个我自认为还行的名字：我自己的世界。（这对于起名困难症来说其实很正常）

随后摘下了头盔，又加入了魔女茶话会，随手拿起了一根巧克力棒，“确实是那些个任务，不过我这边好像是两个都有，一个是实力达到半神，还有一个是带领我的种族进入公国管理层。”我咬了一口拿在手上的巧克力棒，郁闷的说到，“有点难啊……”

“谁让你没自己设置，全靠AI生成，要不然可以简单一点的。”羽陌在一旁打击了我一下，“我设置的仅仅是成为自己门派的领袖而已。”

“emmm，你的门派不是天下第一门派吗？”我有点疑惑地问到。

“是啊，但是也没你那个那么离谱啊，你那个基本等于说你要挑战整个公国所有家族的幕后掌权者，他们手里至少有几个半神吧，吗就是意味着你要同时面对好几个半神，那你的实力要基本上和真神差不多啊。”羽陌一根一根手指点着我需要达到的情况，“还有啊，你懂政治吗？单靠实力可是站不稳的。”

随着她越说越多，我的头瞬间大了起来，于是连忙制止了她接下来的话语。“好了好了，差不多了，先睡觉吧，毕竟明天早上6点就能玩了，今晚早点睡，明天早点起。”

“也对，那这周的魔女茶会就到这吧”，欧阳倩说到，“接下来该去准备准备进入自己的世界里面遨游了！”

于是乎，我们轮流着洗漱，在10点前爬上了床，设好了第二天的闹钟，准备着迎接自己的世界诞生。
